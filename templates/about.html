This tool for Subject Matter Extracting was created as a final project for Computer Science course. The algorithm is based on Frequency Distribution and Trigram methods. The detailed description of the algorithm and code can be found here, though as an original source was pretty much used this article. At the moment the tool works only with English texts, but in the future it will be developed for other languages as well. Any suggestions on how this tool can be improved are very welcome :)
